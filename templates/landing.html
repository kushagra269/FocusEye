// 03 detection systems

Three signals.
One verdict.

Every frame, FocusEye runs a full computer-vision pipeline on your webcam — locally, on CPU. No cloud, no upload, no compromise.

01

Eye Gaze

MediaPipe FaceMesh · 478 landmarks

Tracks iris position relative to your eye socket. A gaze ratio outside 0.28–0.72 = looking away. 15s personal calibration on every session.

0.67s to flag distraction
02

Head Pose

OpenCV solvePnP · 6-point 3D

Maps facial anchors to a 3D model and decomposes rotation into yaw and pitch. Trips on ±20° yaw or ±22° pitch from your baseline.

±20° yaw threshold
03

Phone Use

YOLOv8-nano + Hands + Pose

Three independent signals: object detection, finger-grip pose, and forward-head reading posture. Catches the phone before you notice you reached for it.

3s instant alert mode

# Create venv (Windows: .\Scripts\activate)
$ python -m venv focuseye-env && source focuseye-env/bin/activate
# Install dependencies
$ pip install -r requirements.txt
# Run (admin needed for hosts blocker)
$ sudo python app.py
